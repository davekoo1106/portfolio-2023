Project Overview

Pinnacle is a fictional real estate company that specializes in luxury properties. For this project, I have created an online brochure that would showcase one luxury property, highlighting the features and providing the necessary details for a potential buyer

Introduction

The goal of this project was to create a brochure for a luxury property using Adobe Indesign.

Design Studio

Logo Process

When coming up with ideas for the logo, I wanted it to be simple yet scalable across other company products such as a business card or a letterhead. Therefore, I created ideas using the first letters of the name, 'P' and 'R'

Color Choice

The colors that I chose for this project were tan, black and white. I chose tan as one of the colors for this project because it promotes comfort and stability additionally, it also symbolizes reliability and is overall a great color to choose for a project like this due to its earthy color resemblance

Wireframes

Before creating the brochure, I made wireframes so that I could visualize the layout of the brochure. This helped alot as I was able to understand which layout would be the most effective for showing off the highlights of the property since this project involved lots of images

Final Product

Key Takeaways

Building out Landing Page

Given the time frame of the project, I was able to complete the brochure. However, if time was permitted, I would have taken this project further and created a landing page with other various properties with the incorporation of their own individual brochure that would allow viewers to take a detailed look into each property

Made for Mobile

To take things further, I would further enhance the project optimizing it for mobile. I would start the process by creating out mobile wireframes to see how it would look then create the UI/UX
