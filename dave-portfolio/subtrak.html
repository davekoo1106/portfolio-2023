Project Overview

Subtrak is a subscription tracking app designed to help users keep track of their various subscriptions in one place. With SubTrak, users can easily monitor their subscription payments, renewal dates, and cancellations.

The app offers a user-friendly, simple interface that displays all subscription information in an organized and easy-to-read format. Users can set up reminders for subscription renewals or cancellations, ensuring they don't end up paying for subscriptions they no longer use.

Introduction

The goal of this project was to create a subscription app using react native and tailwind as the css framework with a user friendly interface.

When I was brainstorming apps to create, I knew that I wanted to create an app that would solve a problem. A problem that I faced was not remembering what I was subscribed to and had money going out without realizing. Furthermore, I did some research and found the following:
